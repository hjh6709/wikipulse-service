# WikiPulse — 페이지 구성

---

## 전체 페이지 맵

```
/ (랜딩)                  — 비로그인 접근 가능
/login                   — 로그인
/onboarding              — 첫 로그인 시 관심 카테고리 설정
/issues                  — 트렌딩 이슈 리스트 + 검색바
/issues/[id]             — 이슈 상세
/history                 — 아카이브 / 히스토리 검색
/settings/account        — 계정 정보 (프로필, 비밀번호, 로그아웃)
/settings/preferences    — 관심 카테고리, 구독 토픽 관리
/settings/bookmarks      — 북마크한 이슈 목록
/settings/alerts         — 알림 설정 (Discord / 이메일)
404                      — 없는 페이지
```

---

## 페이지별 상세

### `/` 랜딩 페이지 (비로그인)

- 서비스 소개 및 핵심 차별점
- 실시간 트렌딩 이슈 미리보기 (상위 3~5개)
- "더 보려면 로그인" 유도 버튼

---

### `/onboarding` 온보딩 (첫 로그인 시)

- 관심 카테고리 선택 유도
- 완료 시 `/issues`로 이동
- 건너뛰기 가능 (나중에 `/settings/preferences`에서 변경 가능)

---

### `/issues` 트렌딩 이슈 리스트 (로그인 필요)

- 검색바 — 키워드로 활성 이슈 검색
- 현재 활성 이슈만 표시 (소강 상태는 자동 제거)
- 카테고리 필터 (전체 / 정치 / 경제 / 스포츠 / 연예 / 사회·범죄 / 과학·기술)
- 이슈 카드: 제목, 상태 뱃지, 편집 횟수, 감성 요약, 발생 시각

---

### `/issues/[id]` 이슈 상세 (로그인 필요)

스크롤 구성 — 위에서 아래로 "요약 → 데이터 → 상세" 흐름

```
─────────────────────────────────────────
 이슈 제목
 상태 뱃지: 발생 / 확산 / 정점 / 소강
 데이터 신뢰도 표시 (편집자 수, Reddit 샘플 수)
─────────────────────────────────────────
 AI 브리핑 카드
 "지금 무슨 일이 벌어지고 있는가" 3~5문장 요약
 + 핵심 논점 3개
─────────────────────────────────────────
 편집 스파이크 그래프 (Recharts)
 시간대별 편집 횟수 실시간 업데이트
─────────────────────────────────────────
 감성 분포 차트  |  토픽 클러스터
 긍정/부정/중립       Reddit 댓글 주제별 군집
─────────────────────────────────────────
 이슈 타임라인
 편집 폭증 시점, Reddit 대화량, 감성 변화 흐름
─────────────────────────────────────────
 Reddit 댓글 피드 (실시간)
─────────────────────────────────────────
 관련 종목 (해당되는 경우만 표시)
 이슈 키워드 → 관련 섹터 및 종목
 과거 유사 이슈 발생 시 섹터 변동성 통계
 ※ 본 정보는 투자 조언이 아닙니다
─────────────────────────────────────────
```

**관련 종목 표시 기준:**
- 전 카테고리 대상 (경제만이 아닌 모든 이슈)
- 키워드와 종목 연결이 명확한 경우만 표시
- 연결고리가 없거나 억지스러운 경우 섹션 자체 숨김

---

### `/settings` 사용자 설정 (로그인 필요)

**`/settings/account`** — 계정 정보
- 프로필 (이름, 이메일)
- 비밀번호 변경
- 로그아웃

**`/settings/preferences`** — 개인화
- 관심 카테고리 설정
- 구독 중인 토픽 관리

**`/settings/bookmarks`** — 북마크
- 북마크한 이슈 목록
- 북마크 삭제

**`/settings/alerts`** — 알림 설정
- 알림 받을 이슈/키워드 등록
- 알림 채널 설정 (Discord / 이메일)
- 알림 임계값 설정 (edit_count 기준)

---

### `/history` 히스토리 검색 (로그인 필요)

- 소강/종료된 이슈 아카이브
- 키워드/날짜/카테고리 검색
- 과거 이슈 상세 조회 가능

---

### 에러 / 상태 페이지

- `404` — 없는 페이지 (홈으로 돌아가기 버튼)
- 로딩 스켈레톤 UI — 이슈 리스트, 이슈 상세 각각 적용

---

## 이슈 생애주기

```
발생 → 확산 → 정점 → 소강 → 아카이브
```

| 상태 | 홈 노출 | 상세 접근 |
|---|---|---|
| 발생 / 확산 / 정점 | ✅ | ✅ |
| 소강 | ❌ (자동 제거) | ✅ |
| 아카이브 | ❌ | ✅ (/history) |

---

## 정보 제공 원칙

**WikiPulse는 데이터를 보여줄 뿐, 판단은 사용자가 한다.**

- AI 브리핑은 "~라는 주장이 있습니다", "~측은 ~라고 말합니다" 형태 유지
- 관련 종목은 "관련 종목" + "과거 변동성 통계"만 제공, 투자 권유 표현 금지
- 감성 분석은 Reddit 샘플 기반임을 명시 (편향 가능성 안내)
- 정치 이슈는 특정 성향 판단 금지
- 사회/범죄 이슈는 피의자 추정 보도 금지
